Auteur : Nicolas Le Guerroué
\newcommand{\constrain}[1]{#1} Ce logiciel permettra de gérer efficacement les composants électroniques.
  • Une bibliothèque commune de composants
  • Une bibliothèque personnelles
  • Un identifiant unique
    • \constrain{PRIMARY KEY}
  • Une référence unique
    • \constrain{ALPHANUM} \constrain{UPPERCASE}
  • Une désignation
  • Une catégorie
  • Une sous-catégorie
  • Un boîtier
  • Une plage d'alimentation
  • Une documentation technique
  • Une image
Les images et documents ne seront pas stockés au format binaire dans la base de données, cela prend trop de place et ralentis considérablement le temps d'accès aux données.

Cas d'usages

  • L'utilisateur souhaite ajouter un composant non existant dans la bibliothèque
  • L'utilisateur souhaite ajouter un composant dans sa base de données personnelle.
    • Le composant existe, il l'ajoute dans son espace personnel.
    • Le composant n'existe pas dans la bibliothèque, il doit l'ajouter manuellement dans la bibliothèque puis l'ajouter dans son espace personnel.
  • L'utilisateur souhaite consulter un composant dans la bibliothèque
  • L'utilisateur souhaite consulter un composant dans son espace personnel
